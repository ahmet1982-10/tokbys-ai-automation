# 🇺🇸 English

## Overview

TOKBYS AI Automation Suite is a desktop automation platform designed to reduce repetitive work performed during cooperative credit collection operations.

Instead of manually navigating through multiple screens, the application assists the operator by automating repetitive workflow steps while preserving human supervision when required.

The project was iteratively improved with **OpenAI Codex** during **OpenAI Build Week 2026**.

---

## Why this project?

This project has been used in daily cooperative credit collection operations since March 25, 2026, and has continued to evolve through real operational feedback.

During OpenAI Build Week, OpenAI Codex was used to refactor, improve and extend the system with new capabilities.

---

## Features

- Automated credit search
- Collection workflow automation
- Receipt generation
- Hybrid DOM + coordinate-based automation
- JavaScript-assisted navigation
- Retry mechanism
- Resume interrupted workflow
- Intelligent workflow state detection
- Modular Python architecture

# 🇹🇷 Türkçe

## Proje Hakkında

TOKBYS AI Automation Suite, kooperatif kredi tahsilat işlemlerinde operatörün tekrar eden iş yükünü azaltmak amacıyla geliştirilmiş masaüstü otomasyon sistemidir.

Sistem;

- kredi arama işlemlerini kolaylaştırır,
- tahsilat iş akışını otomatikleştirir,
- dekont oluşturma sürecini hızlandırır,
- JavaScript destekli gezinme ile işlem adımlarını yönetir,
- gerektiğinde koordinat tabanlı işlemleri kullanır,
- hata oluştuğunda güvenli şekilde tekrar deneyebilir,
- yarım kalan işlemlere devam edebilir.

Kodun önemli bölümleri OpenAI Codex ile birlikte geliştirilmiş ve iyileştirilmiştir.
